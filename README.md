# Valoria - Portefeuille d'Actifs Personnel 🥂

Valoria est une application Android moderne conçue pour suivre la performance de vos investissements (Actions, ETF, Cryptomonnaies, Métaux précieux) avec une interface élégante et des fonctionnalités de synchronisation flexibles.

## 🚀 Fonctionnalités Clés

- **Tableau de Bord Holistique** : Vue d'ensemble de la valeur totale, profit/perte total et quotidien.
- **Support Multi-Actifs** : 
  - 📈 **Actions & ETF** (Yahoo Finance)
  - ₿ **Cryptomonnaies** (CoinGecko)
  - 🥇 **Métaux Précieux** (Or, Argent - Napoléons, Lingots).
- **Graphiques de Performance Interactifs** :
  - Filtres temporels : 24h, 7j, 1m, 1an, 5ans, Tout (depuis 2015).
  - Points d'achat (verts) et de vente (rouges) affichés directement sur la courbe.
  - Détails au survol avec date et variation.
- **Système d'Alertes Intelligent** :
  - Notifications automatiques si un actif varie de plus de **3%** sur la journée.
  - Surveillance en arrière-plan toutes les 2 heures.
- **Souveraineté des Données** :
  - **Auto-Sync** : Synchronisation automatique via un dossier local (compatible avec Google Drive, Syncthing, etc.).
  - **Import/Export** : Format de fichier `.val` simple pour la portabilité.
  - **Confidentialité** : Pas de compte requis, les données restent sur votre appareil.

## 🛠️ Stack Technique

- **Langage** : Kotlin
- **UI** : Jetpack Compose avec un design "Premium Dark/Gold".
- **Architecture** : MVVM (Model-View-ViewModel) avec Clean Architecture.
- **Injection de dépendances** : Hilt.
- **Base de données** : Room.
- **Réseau** : Retrofit & Gson.
- **Tâches de fond** : WorkManager.
